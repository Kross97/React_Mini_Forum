# React_Mini_Forum (React / PHP) 1\3 серии форумов (fullstack)
(Технологии:
1.@reduxjs/toolkit(createEntityAdapter, createAsyncThunk),
2.normalizr.js,
3.Formik,
4.Yup,
5.styled-components,
6.i18next,
7.redux-persist,
8.redux-batched-actions )
